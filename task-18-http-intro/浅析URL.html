URL 包含哪几部分，每部分分别有什么作用
            URL 统一资源定位符
            uniform resource locator
                以下面的URL为例
                    https://www.baidu.com/s?wd=hello&rsv_spt=1#5
                协议 确定URL使用什么传输协议 如http https
                域名或IP 确定URL的访问地址
                (端口号) 确定服务的端口号
                路径 用来请求同一个网站下的不同页面
                查询字符串 获取同一个页面的不同显示内容
                锚点 同一个页面 同一个查询参数 不同显示位置 锚点可以做到

        DNS 的作用是什么
            Domain Name System
            域名和IP如何对应起来

        nslookup 命令怎么用
            nslookup用于查询DNS的记录，查询域名解析是否正常，在网络故障时用来诊断网络问题
            nslookup baidu.com

        IP 的作用是什么
            internet protocol
            没有IP 不能上网
            主要约定了两件事
                1 如何定位一台设备
                2 如何封装数据报文 跟其他设备交流
            只要处在互联网中 至少拥有一个独立的IP 
            IP分为内网和外网

        ping 命令怎么用
            ping 域名 或 IP地址

        域名是什么，分别哪几类域名
            以www.xiedaimala.com为例
                com 一级域名
                xiedaimala.com 二级域名
                www.xiedaimala.com 三级域名
                www.xiedaimala.com和xiedaimala.com可以不是同一家公司
            www是多余的
